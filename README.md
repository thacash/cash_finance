Tools for options sellers.

## Install

```
$ npm install @thacash/cash_finance
```

## Usage

```js
const cashfinance = require("@thacash/cash_finance");

cashfinance.getContractIncomePercent(contractPrice, stockPrice);
//=> 28.79

```
